Presentation mode turns your annotated campaign data into a structured, slide-by-slide walkthrough. It is designed to help Red Teams communicate findings to customers and stakeholders.

## Opening Presentation mode

Click the **presentation** icon in the navigation bar to enter Presentation mode.

## Presentation list

The first screen shows a list of available presentations:

- **All** — A collection of every comment in the campaign.
- **Favorite** — A collection of all comments that have been marked as favorites.
- **Tag presentations** — One presentation for each tag, containing only the comments associated with that tag.

<Info>
  Presentations are automatically generated from your comments and tags. No manual setup is required beyond adding comments and favoriting them.
</Info>

## Slideshow view

Clicking a presentation in the list opens the slideshow. Each slide represents a comment tied to a command, and slides are ordered by the time of the associated command.

### Navigating slides

| Control | Action |
|---|---|
| **Next** button | Move forward to the next slide |
| **Previous arrow** (left of Next) | Move back to the previous slide |
| Slide indicator | Click a position in the indicator to jump to a specific slide |
| **Left arrow** (far left of panel) | Return to the presentations list |

<Tip>
  Use the **Favorite** presentation to build a curated, high-priority walkthrough. Mark the most significant findings as favorites while reviewing the campaign, then present using the Favorite collection.
</Tip>

## How comments become slides

Every comment you add to a command becomes a potential slide in a presentation. To make the most of Presentation mode:

1. Add comments to significant commands during campaign review. See [Comments and tags](/guides/comments-and-tags).
2. Favorite the most important comments to include them in the **Favorite** presentation.
3. Use tags to organize findings by theme — each tag automatically creates its own presentation.
