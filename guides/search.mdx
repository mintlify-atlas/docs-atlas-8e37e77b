title: "Search"
description: "Full-text search across beacons, hosts, operators, comments, commands, and more within a campaign."
---

Search lets you find any item in a campaign quickly without having to navigate through the Explorer tabs manually.

## Opening search

Search is available from the **eyeglass** icon in the navigation bar, or with a keyboard shortcut from anywhere within a campaign:

| Platform | Shortcut |
|---|---|
| Windows | `Ctrl + K` |
| Linux | `Ctrl + K` |
| Mac | `Cmd + K` |

## What you can search

Search performs full-text matching across all of the following item types:

## Filtering and sorting results

**Filter by type** — Narrow results to any single item type from the list above.

**Sort results** by:

| Sort option | Order |
|---|---|
| Relevance | Ascending or descending |
| Name | Ascending or descending |
| Type | Ascending or descending |
