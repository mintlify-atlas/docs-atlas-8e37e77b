## What is RedEye?

RedEye is an open-source analytic tool developed by [CISA](https://www.cisa.gov/) and [DOE](https://www.energy.gov/)'s [Pacific Northwest National Laboratory](https://www.pnnl.gov/) to assist Red Teams with visualizing and reporting command-and-control (C2) activities.

RedEye parses logs from C2 frameworks like Cobalt Strike, Brute Ratel, and Sliver, then presents that data in an interactive graph-based interface. Operators can tag and annotate commands, replay attack timelines, and export curated campaign data for Blue Team review.

## Red Team and Blue Team modes

RedEye operates in two distinct modes covering both stages of a Red Team engagement:

| Feature | Red Team Mode | Blue Team Mode |
|---|---|---|
| Upload C2 logs | ✓ | ✗ |
| Edit and annotate | ✓ | ✗ |
| Create presentations | ✓ | ✓ |
| Export campaigns | ✓ | ✓ |
| Authentication required | ✓ (password) | ✗ |

<Note>
  Both modes run from the same RedEye binary. Pass `--redTeam --password <password>` to enable Red Team mode, or run without arguments for Blue Team mode.
</Note>

## Supported C2 frameworks

RedEye ships with built-in parsers for the most common C2 frameworks:

- **Cobalt Strike** — Full beacon log parsing including metadata, commands, and output
- **Brute Ratel** — Campaign log parsing and visualization
- **Sliver** — Log ingestion and visualization

You can also [build a custom parser](/parsers/creating-a-parser) for any C2 framework.

## Get started

<Steps>
  <Step title="Download RedEye">
    Download the latest binary for your OS from the [GitHub Releases](https://github.com/cisagov/RedEye/releases) page.
  </Step>
  <Step title="Start the server">
    Run `./RedEye --redTeam --password <your_password>` for Red Team mode, or simply `./RedEye` for Blue Team mode.
  </Step>
  <Step title="Open the web app">
    Navigate to [http://127.0.0.1:4000](http://127.0.0.1:4000) in your browser (RedEye opens it automatically).
  </Step>
  <Step title="Import a campaign">
    Upload your C2 logs or try the [example dataset](https://github.com/cisagov/RedEye/raw/develop/applications/redeye-e2e/src/fixtures/gt.redeye) to get started.
  </Step>
</Steps>
